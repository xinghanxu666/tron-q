# tron_address

波场地址查询插件

## 功能

- 查询波场地址的TRX余额、USDT余额
- 查询账户激活时间
- 查询交易次数
- 查询带宽和能量使用情况
- 查询投票情况

## 使用说明

命令格式: `/tron [波场地址]`
示例: `/tron TBQpETG8uJrktewhv832m7wSKmvMUUsZCm`

帮助命令: `/tron.help`

## 配置

在使用前，请确保在插件配置中填写您的TRONGRID API密钥。

1. 在AstrBot管理面板中，找到插件管理。
2. 点击`tron_address`插件的配置。
3. 在`TRONGRID API密钥`一栏中填入您的密钥（在https://trongrid.io/ 注册并获取）。
4. 保存配置。
